# Hi, I'm Noel

I build practical software for businesses, developers, and Linux desktop workflows.

Most of my work is around:

- **Web apps** with TypeScript, React, Next.js, and Node.js
- **AI-assisted tools** for automation, productivity, and developer workflows
- **Linux desktop utilities** for Hyprland, dual-boot setups, and everyday tooling
- **Client projects** where speed, reliability, and maintainability matter more than hype

## Current focus

- Building useful AI tools that solve narrow, real problems
- Improving my open-source projects so they are easier to install, test, and contribute to
- Shipping cleaner product demos, documentation, and deployment workflows

## Selected projects

- [`voxassist`](https://github.com/ziuus/voxassist) - AI-assisted clinical note generation from doctor-patient conversations
- [`lindy`](https://github.com/ziuus/lindy) - Linux desktop app for accessing Windows user folders from a dual-boot setup
- [`waiting-game`](https://github.com/ziuus/waiting-game) - Lightweight transparent overlay game for killing time during long-running tasks
- [`Synapse`](https://github.com/ziuus/Synapse) - Local agent framework experiments with routing, telemetry, and desktop automation

## Tech I use often

TypeScript · JavaScript · React · Next.js · Node.js · Python · Rust · Linux · Docker · GitHub Actions
